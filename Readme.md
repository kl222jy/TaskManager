TaskManager
-------------------------------
Uppgiftshanterare ur projekthanteringssystem

I katalogen doc hittar du aks som beskriver databasen i helhet, detta gränssnitt kommer att bearbeta data i tabellerna Task, MemberTask och ProjectMember.
Data från övriga tabeller kommer i viss mån att hämtas ut och till viss del hårdkodas för att få någon rimlig data att arbeta med.

Att medlemmen 3 har ändrat status på en uppgift till 2 i projektet 1 blir lite väl kryptiskt.
